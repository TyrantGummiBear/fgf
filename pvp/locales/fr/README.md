## Règles rapides

| Situation | Règle |
|---------------|------|
| **Positionnement par défaut** | **Combattez autour de nos baleines** — elles sont l'ancre ; la flotte reste sur eux |
| **Notre baleine contre la baleine ennemie** | **Jamais seul** — le support doit se joindre immédiatement |
| **Combat de baleines en cours** | Aidez **nos** baleines d'abord → concentrez-vous sur les baleines ennemies → puis sur les attaquants individuels |
| **PvP à grande échelle** | **Éliminez d'abord les menaces** — ne poursuivez pas les victimes faciles pendant que les cibles dangereuses restent en alerte |
