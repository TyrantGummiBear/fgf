## Быстрые правила

| Ситуация | Правило |
|-----------|------|
| **Позиционирование по умолчанию** | **Сражайтесь вокруг наших китов** — они якорь; флот остается на них |
| **Наш кит против вражеского кита** | **Никогда не одинок** — служба поддержки должна присоединиться немедленно |
| **Идет борьба с китами** | Сначала помогите **нашим** китам → сосредоточьте внимание на вражеских китах → затем на отдельных нападающих |
| **Масштабное PvP** | **Сначала устраните угрозы** — не гонитесь за легкими убийствами, пока опасные цели не спят |
