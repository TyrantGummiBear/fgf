## Reglas rápidas

| Situación | Regla |
|-----------|--------------|
| **Posicionamiento predeterminado** | **Lucha alrededor de nuestras ballenas**: ellas son el ancla; la flota permanece sobre ellos |
| **Nuestra ballena vs ballena enemiga** | **Nunca solo**: el soporte debe unirse de inmediato |
| **Lucha de ballenas en progreso** | Ayuda a **nuestras** ballenas primero → enfoca a las ballenas enemigas → luego a los atacantes individuales |
| **PvP a gran escala** | **Elimina las amenazas primero**: no persigas asesinatos fáciles mientras los objetivos peligrosos permanecen despiertos |
