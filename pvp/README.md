## Quick rules

| Situation | Rule |
|-----------|------|
| **Default positioning** | **Fight around our whales** — they are the anchor; the fleet stays on them |
| **Our whale vs enemy whale** | **Never alone** — support must join immediately |
| **Whale fight in progress** | Help **our** whales first → focus enemy whales → then individual attackers |
| **Large-scale PvP** | **Eliminate threats first** — don't chase easy kills while dangerous targets stay up |
