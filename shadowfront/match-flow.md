# Match Flow

Priorities by phase. Throughout: **win the number fight in the capture circle.**

---

## Early Game (First ~5 min)

**Goal:** Secure a home vault, scout enemy stacks, don't throw HP.

| Do | Don't |
|----|-------|
| Stack numbers in an open circle | Solo push a defended vault |
| Get bodies inside before fighting outside | Assume one strong fleet holds |
| Assign home hold squad | Split across every vault |

**Coordinator focus:** One attack vault, stack for count advantage.

---

## Mid Game

**Goal:** Bonus windows, hold counts, rotate as a team.

| Do | Don't |
|----|-------|
| Remove enemies **in the circle** when they're capturing | Farm damage outside the zone |
| Call **Flip** vs **Deny** — [Bonus Points](bonus-points.md) | Force flip when count is unwinnable |
| Stack before tower push if outnumbered | Drop tower into a losing count |
| Nodes → back in circle | Stay out "recovering" |

**Coordinator focus:** Vault priority, number advantage, flip/deny timing.

---

## Late Game

**Goal:** Deny leader vaults, protect score, no throw plays.

| Do | Don't |
|----|-------|
| **Deny** when we can't win capture | Commit to flips we can't hold |
| Stack numbers on the vault that decides the match | Spread for damage stats |
| Flex squad for late denies | Leave home circle empty |

**Coordinator focus:** Leaderboard math — which vault and which mode (flip vs deny)?

---

## Phase Checklist by Role

### Stack / Capture
- Early: Win count in called vault circle
- Mid: Hold number advantage through bonus windows
- Late: All-in on win-condition vault

### Home / Hold
- Early: Don't lose home count to a free walk-in
- Mid: Call stack before count flips
- Late: Home circle may be the win — keep bodies in it

### Flex / Deny
- Early: Reinforce stacks, build numbers
- Mid: Flip if count winnable; deny if not
- Late: Save HP for final deny breaks

---

## Same-Server Alliances

- Confirm in chat who you're not fighting
- Alliance doesn't mean free captures — still need **numbers in the circle**
- Coordinator updates if alliance breaks

---

## Summary

| Phase | Priority |
|-------|----------|
| Early | Stack numbers, take one vault |
| Mid | Circle fights + bonus flip/deny |
| Late | Deny leader or hold winning count |
