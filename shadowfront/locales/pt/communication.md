## Chamadas do coordenador (todos ouvem)

| Ligue | Significado |
|------|---------|
| **"Pilha [cofre]"** | Entre, obtenha números **dentro do círculo** |
| **"Segure [cofre]"** | Defenda — mantenha a contagem em nosso círculo |
| **"Girar para [cofre]"** | Sair, agrupar no novo círculo do vault |
| **"Virar [cofre]"** | Ganhe números, roube bônus - [Pontos de bônus](bonus-points.md) |
| **"Negar [cofre]"** | Evite bônus – não precisa segurar a captura |
| **"Guardião [cofre]"** / **"Torre [cofre]"** | Guardião do cofre de foco (recaptura de blocos) |
| **"Guardião primeiro [cofre]"** / **"Torre primeiro [cofre]"** | Grande entrada – solte o guardião e depois lute pela contagem |
| **"Guardião caído [cofre]"** | Recapturar aberto – círculo de pilha agora |
| **"Preciso de ajuda [cofre]"** | Estamos perdendo a luta pelos números |
| **"Nós para cima [cofre]"** | Pegue os nós de reparo, de volta ao círculo |
| **"Tarde demais para virar"** | Negar apenas |
| **"Casa!"** | Inimigo em nosso círculo — empilhe agora |
| **"Isca [cofre]"** | Pipas de navio nomeadas - todos os outros **mantenha o círculo** |
| **"Não persiga"** | Círculo esquerdo do corredor – fique na contagem |
| **"Foco [nome]"** / **"HP baixo [nome]"** | Queime o alvo de HP mais baixo em círculo |
| **"Baleia em foco [nome]"** | Queime a baleia inimiga — **somente quando tivermos números** |
| **"Evite baleia"** | Não negocie; pipa se for alvo, isca se eles perseguirem |
| **"Pilha em [nome]"** | Grupo em nossa baleia dentro do círculo |

## Chamadas de jogadores (seja breve)

| Ligue | Quando |
|------|------|
| **"Em círculo [cofre]"** | Você está dentro da zona de captura |
| **"Eles têm números [cofre]"** | Inimigo vencendo a contagem |
| **"Contagem par [cofre]"** | Contestado — preciso de mais um corpo |
| **"Não é possível segurar [cofre]"** | Perdendo a conta, preciso de pilha |
| **"Torre derrubada [cofre]"** | O mesmo que guardião abatido – insira os números agora |
| **"Contagem regressiva [cofre] — X à esquerda"** | Tempo para inverter/negar |
| **"Isca [cofre]"** | Você está se arrastando – companheiros de equipe contam |
| **"Sou alvo"** | Tomando o foco – pode ser necessário fazer um micro out |
| **"Baleia em mim"** | Baleia foca em você – evite/arraste |

## O que NÃO dizer

- ❌ "Estou causando um bom estrago aqui"
- ❌ "Minha frota é forte o suficiente sozinha"
- ❌ "Perseguindo [nome]" (fora do círculo)

Se você está dizendo isso, provavelmente **não está ganhando a contagem**.
