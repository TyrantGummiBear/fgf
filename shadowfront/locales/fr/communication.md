#Communication

Appels courts uniquement. Les messages longs sont ignorés en plein combat.

## Appels du coordinateur (tout le monde écoute)

| Appeler | Signification |
|------|--------------|
| **"Pile [coffre-fort]"** | Emménagez, obtenez des chiffres **à l'intérieur du cercle** |
| **"Maintenir [coffre-fort]"** | Défendre – comptez dans notre cercle |
| **"Tourner vers [coffre-fort]"** | Départ, groupe au nouveau cercle de coffre-fort |
| **"Retourner [coffre-fort]"** | Gagnez des numéros, volez des bonus — [Points bonus](bonus-points.md) |
| **"Refuser [coffre-fort]"** | Prévenir le bonus — pas besoin de maintenir la capture |
| **"Gardien [coffre-fort]"** / **"Tour [coffre-fort]"** | Focus Vault Guardian (bloque la recapture) |
| **"Le gardien en premier [coffre-fort]"** / **"La tour d'abord [coffre-fort]"** | Gros entrant – déposez le gardien, puis combattez pour le compte |
| **"Gardien en bas [coffre-fort]"** | Re-capturer ouvert — empiler le cercle maintenant |
| **"Besoin d'aide [coffre-fort]"** | Nous perdons la bataille des chiffres |
| **"Nœuds en place [coffre-fort]"** | Saisissez les nœuds de réparation, revenez en cercle |
| **"Trop tard pour retourner"** | Refuser uniquement |
| **"Maison!"** | Ennemi dans notre cercle — empilez maintenant |
| **"Appâter [coffre-fort]"** | Cerfs-volants de navire nommés — tous les autres **tenez le cercle** |
| **"Ne poursuivez pas"** | Cercle gauche du coureur – restez sur le compte |
| **"Concentrez-vous sur [nom]"** / **"Faible HP [nom]"** | Brûler la cible HP la plus basse en cercle |
| **"Concentrez-vous sur la baleine [nom]"** | Brûlez la baleine ennemie — **uniquement lorsque nous avons des chiffres** |
| **"Éviter les baleines"** | N'échangez pas ; cerf-volant s'il est ciblé, appât s'il poursuit |
| **"Empiler sur [nom]"** | Groupe sur notre baleine à l'intérieur du cercle |

## Appels de joueurs (Soyez bref)

| Appeler | Quand |
|------|------|
| **"En cercle [coffre-fort]"** | Vous êtes à l'intérieur de la zone de capture |
| **"Ils ont des numéros [coffre-fort]"** | L'ennemi remporte le décompte |
| **"Compte pair [coffre-fort]"** | Contesté – besoin d’un corps supplémentaire |
| **"Impossible de conserver [coffre-fort]"** | Perdre le compte, besoin de pile |
| **"Tour vers le bas [coffre-fort]"** | Identique au gardien vers le bas – entrez les chiffres maintenant |
| **"Compte à rebours [coffre-fort] — X gauche"** | Moment pour retourner/refuser |
| **"Appâtage [voûte]"** | Vous traînez — vos coéquipiers comptent |
| **"Je suis ciblé"** | Se concentrer – il faudra peut-être faire un micro |
| **"Baleine sur moi"** | Baleine, concentrez-vous sur vous — évitez/traînez |

---

## Ce qu'il ne faut PAS dire

- ❌ "Je fais de bons dégâts ici"
- ❌ "Ma flotte est assez forte en solo"
- ❌ "Chasing [nom]" (en dehors du cercle)

Si vous dites cela, vous ne gagnez probablement **pas le décompte**.
