Только короткие выноски. Длинные сообщения игнорируются во время боя.

## Звонок координатора (все слушают)

| Позвонить | Значение |
|------|---------|
| **"Стек [хранилище]"** | Подойдите, получите цифры **внутри круга** |
| **"Держать [хранилище]"** | Защити — держи счёт в нашем кругу |
| **"Повернуть в [хранилище]"** | Уходите, группа в новом круге хранилища |
| **"Перевернуть [хранилище]"** | Выигрывайте номера, крадите бонус — [Бонусные баллы](bonus-points.md) |
| **"Запретить [хранилище]"** | Запретить бонус — не нужно удерживать захват |
| **"Страж [хранилище]"** / **"Башня [хранилище]"** | Страж хранилища фокуса (блокирует повторный захват) |
| **"Сначала Страж [хранилище]"** / **"Сначала Башня [хранилище]"** | Большой входящий — сбросьте Guardian, затем сражайтесь за счет |
| **"Страж упал [хранилище]"** | Повторный захват открыт — теперь сложите круг |
| **"Нужна помощь [хранилище]"** | Мы проигрываем числовой бой |
| **"Узлы подняты [хранилище]"** | Возьмите ремонтные узлы и снова по кругу |
| **"Слишком поздно переворачивать"** | Запретить только |
| **"Домой!"** | Враг в нашем кругу — теперь стек |
| **"Выманить [хранилище]"** | Именованные корабельные змеи — все остальные **держат круг** |
| **"Не гонись"** | Бегун покинул круг — оставайся на счету |
| **"Фокус [имя]"** / **"Низкое здоровье [имя]"** | Сжечь цель с самым низким HP в круге |
| **"Фокус кит [имя]"** | Сжечь вражеского кита — **только когда у нас есть цифры** |
| **"Избегайте китов"** | Не торгуйте; воздушный змей, если на него напали, приманка, если преследуют |
| **"Стек на [имя]"** | Группа на нашем ките внутри круга |

## Звонки игрока (будьте краткими)

| Позвонить | Когда |
|------|------|
| **"В круге [свод]"** | Вы находитесь в зоне захвата |
| **"У них есть номера [хранилище]"** | Враг выигрывает счет |
| **"Даже считать [хранилище]"** | Оспаривается — нужен еще один орган |
| **"Не могу удержать [хранилище]"** | Теряем счет, нужен стек |
| **"Башня разрушена [хранилище]"** | То же, что и при отключении Guardian — введите цифры сейчас |
| **"Обратный отсчет [хранилище] — осталось X"** | Время для переворота/запрета |
| **"Травля [хранилище]"** | Вы затягиваете — товарищи по команде держат счет |
| **"Я стал мишенью"** | Фокусируемся — возможно, потребуется микровыход |
| **"Кит на мне"** | Китовый фокус на тебе — избегать/затягивать |

## Чего НЕ говорить

- ❌ «Я наношу здесь хороший урон»
- ❌ «Мой флот достаточно силен в одиночку»
- ❌ «Погоня за [имя]» (вне круга)

Если вы говорите это, вы, вероятно, **не выигрываете**.
