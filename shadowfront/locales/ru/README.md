Краткий справочный центр по совпадениям Foundation: Galactic Frontier **ShadowFront**.

**Думайте об этом как о CoD Control.** В каждом хранилище есть **круг захвата**. Флоты внутри круга учитываются — **побеждает большинство чисел**, а не самые сильные корабли.

## Единственное правило

> **Если вы не в кругу захвата, вы не оспариваете эту точку зрения.**

Capture — это **игра с числами**. Сложите круг, удалите из него врагов и продолжайте считать.
