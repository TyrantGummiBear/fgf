# Communication

Short callouts only. Long messages get ignored mid-fight.

---

## Coordinator Calls (Everyone Listens)

| Call | Meaning |
|------|---------|
| **"Stack [vault]"** | Move in, get numbers **inside the circle** |
| **"Hold [vault]"** | Defend — keep count in our circle |
| **"Rotate to [vault]"** | Leave, group at new vault circle |
| **"Flip [vault]"** | Win numbers, steal bonus — [Bonus Points](bonus-points.md) |
| **"Deny [vault]"** | Prevent bonus — don't need to hold capture |
| **"Tower first [vault]"** | Big incoming — drop tower, then fight for count |
| **"Need help [vault]"** | We're losing the number fight |
| **"Nodes up [vault]"** | Grab repair nodes, back in circle |
| **"Too late to flip"** | Deny only |
| **"Home!"** | Enemy in our circle — stack now |
| **"Bait out [vault]"** | Named ship kites — everyone else **hold circle** |
| **"Don't chase"** | Runner left circle — stay on count |
| **"Focus [name]"** / **"Low HP [name]"** | Burn lowest HP target in circle |
| **"Focus whale [name]"** | Burn enemy whale — **only when we have numbers** |
| **"Avoid whale"** | Don't trade; kite if targeted, bait if they chase |
| **"Stack on [name]"** | Group on our whale inside circle |

---

## Player Calls (Keep It Short)

| Call | When |
|------|------|
| **"In circle [vault]"** | You're inside the capture zone |
| **"They have numbers [vault]"** | Enemy winning the count |
| **"Even count [vault]"** | Contested — need one more body |
| **"Can't hold [vault]"** | Losing count, need stack |
| **"Tower down [vault]"** | Push numbers in now |
| **"Countdown [vault] — X left"** | Timing for flip/deny |
| **"Baiting [vault]"** | You're dragging out — teammates hold count |
| **"I'm targeted"** | Taking focus — may need to micro out |
| **"Whale on me"** | Whale focus on you — avoid / drag out |

---

## What NOT to Say

- ❌ "I'm doing good damage out here"
- ❌ "My fleet is strong enough solo"
- ❌ "Chasing [name]" (outside the circle)

If you're saying these, you're probably **not winning the count**.
